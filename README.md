# 工商注销平台 (GongShangZhuXiao)

企业工商注销一站式服务平台。

## 项目简介

帮助企业完成工商注销流程的一站式服务平台，解决注销流程复杂、耗时的问题。

- **目标用户**：需要注销企业的企业主、代理记账公司
- **核心功能**：注销流程指引、材料准备、进度跟踪、在线预约
- **部署状态**：规划中

## 技术栈

- **前端**：Vue 3 + Vite + TypeScript
- **后端**：Node.js + Python
- **数据库**：PostgreSQL + Redis

## 端口规划（40360-40379）

| 端口 | 服务 | 环境 | 描述 |
|-----|------|------|------|
| 40360 | web | dev | Web主站 |
| 40361 | api | dev | API接口服务 |

## 核心功能

1. **注销流程指引** - 清晰的注销步骤说明
2. **材料准备** - 所需材料清单及模板下载
3. **进度跟踪** - 实时查看注销进度
4. **在线预约** - 预约工商窗口办理时间
5. **代理服务** - 代理注销服务（可选）
